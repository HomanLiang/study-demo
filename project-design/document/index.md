3. 常用的库

   1. [日志库](https://github.com/HomanLiang/study-demo/blob/main/project-design/document/chapter6_03_01.md)

   2. [Json库](https://github.com/HomanLiang/study-demo/blob/main/project-design/document/chapter6_03_02.md)

   3. [Activiti](https://github.com/HomanLiang/study-demo/blob/main/project-design/document/chapter6_03_03.md)

   4. [EasyExcel](https://github.com/HomanLiang/study-demo/blob/main/project-design/document/chapter6_03_04.md)

      一款快速、简单避免 OOM 的 java 处理 Excel 工具。

   5. [XXL-JOB](https://www.xuxueli.com/xxl-job/)

   6. [Druid](https://github.com/HomanLiang/study-demo/blob/main/project-design/document/chapter6_03_06.md)

   7. OSHI - 一款为 Java 语言提供的基于 JNA 的（本机）操作系统和硬件信息库

   8. Hutool -  一个非常实用的 Java 工具类库，对文件、流、加密解密、转码、正则、线程、XML 等 JDK 方法进行了封装。

   9. Guava

      Guava工程包含了若干被Google的 Java项目广泛依赖 的核心库，例如：集合 [collections] 、缓存 [caching] 、原生类型支持 [primitives support] 、并发库 [concurrency libraries] 、通用注解 [common annotations] 、字符串处理 [string processing] 、I/O 等等。

   10. MapStruct - Java 实体映射工具

   11. javafaker - 生成仿真数据

   12. Commons BeanUtils

       针对 Bean 的一个工具集。由于 Bean 往往是由一堆 get 和 set 组成，所以 BeanUtils 也是在此基础上进行一些包装。

       一个比较常用的功能是 Bean Copy，也就是 copy bean 的属性。如果做分层架构开发的话就会用到，比如从 PO（Persistent Object）拷贝数据到 VO（Value Object）。

      13. Commons Codec

          是编码和解码组件，提供常用的编码和解码方法，如 DES、SHA1、MD5、Base64、URL 和 Soundx 等。

      14. Commons Collections

          是一个集合组件，扩展了 Java 标准 Collections API，对常用的集合操作进行了很好的封装、抽象和补充，在保证性能的同时大大简化代码。

      15. Commons Compress

          是一个压缩、解压缩文件的组件，可以操作 rar、cpio、Unix dump、tar、zip、gzip、XZ、Pack200 和 bzip2 格式的压缩文件。

      16. Commons Configuration

          是一个 Java 应用程序的配置管理工具，可以从 properties 或者 xml 文件中加载配置信息。

      17. Commons CSV

          是一个用来读写各种 Comma Separated Value（CSV）格式文件的 Java 类库。

      18. Commons Daemon

          实现将普通的 Java 应用变成系统的后台服务，例如 Tomcat 就是利用这个项目来实现作为 Linux 和 Windows 的服务启动和停止的。

      19. Commons DBCP

          数据库连接池。

      20. Commons DBUtils

          是 JDBC 工具组件，对传统操作数据库的类进行二次封装，可以把结果集转化成 List。

      21. Commons Digester

          是 XML 到 Java 对象的映射工具集。

      22. Commons Email

          是邮件操作组件，对 Java Mail API 进行了封装，提供了常用的邮件发送和接收类，简化邮件操作。该组件依赖 Java Mail API。

      23. Commons Exec

          提供一些常用的方法用来执行外部进程，如执行 exe 文件或命令行。

      24. Commons FileUpload

          为 Web 应用程序或 Servlet 提供文件上传功能，Struts2 和 SpringMVC 的文件上传组件。

      25. Commons IO

          是处理 IO 的工具类包，对 java.io 进行扩展，提供了更加方便的 IO 操作。

      26. Commons JCI

          提供通用的 Java 编译器接口。

      27. Commons Lang3

          是处理 Java 基本对象方法的工具类包，该类包提供对字符、数组等基本对象的操作，弥补了 java.lang api 基本处理方法上的不足。

      28. Commons Launcher

          可以跨平台独立启动的 java 应用程序。

      29. Commons Logging

          提供统一的日志接口，同时兼顾轻量级和不依赖于具体的实现。类包给中间件/日志工具开发者一个简单的日志操作抽象，允许程序开发人员使用不同的具体日志实现工具。

      30. Commons Math

          轻量级自容器的数学和统计计算方法类包，包含大多数常用的数值算法。

      31. Commons Net

          封装了各种网络协议的客户端，支持 FTP、NNTP、SMTP、POP3、Telnet 等协议。

      32. Commons Pool

          提供了一整套用于实现对象池化的框架，以及若干各具特色的对象池实现，可以有效地减少处理对象池化时的工作量。类包用于提高像文件句柄、数据库连接、socket 通信这类大对象的调用效率，简单的说就是一种对象一次创建多次使用的技术。

      33. Commons Primitives

          提供了一个更小、更快和更易使用的对 Java 基本类型的支持。

      34. Commons Validator

          提供了一个简单的、可扩展的框架来在一个 XML 文件中定义校验器（校验方法）和校验规则。支持校验规则和错误消息的国际化。
